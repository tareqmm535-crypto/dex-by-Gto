# ديكس++ (Dex++) v3.0

**أداة التصحيح المتقدمة لألعاب روبلوكس - Roblox Debugging Suite**

ديكس++ هو إحياء لأداة Moon's Dex الشهيرة، تم تطويره بواسطة **Chillz** ليكون أداة تصحيح شاملة ومتكاملة لمطوري ألعاب روبلوكس. يدعم جميع منفذي الأوامر (Executors) تقريباً مع آليات احتياطية ذكية.

---

## ✨ المميزات

### 🔍 المستكشف (Explorer)
- تصفح كامل لشجرة الأجزاء (Instance Tree) في اللعبة
- بحث متقدم داخل المساحة (Workspace) والمثيلات الفارغة (Nil Instances)
- قائمة سياقية كاملة: قص، نسخ، لصق، حذف، إعادة تسمية، تجميع، فك تجميع، تحديد الأطفال، الانتقال للأصل
- عرض المسار الكامل ونسخه
- إدراج كائنات جديدة
- حظر/إلغاء حظر الإرسال عن بُعد (Remote Blocking)
- عرض وتفريغ الدوال (Dump Functions)
- تشغيل ClickDetector، TouchTransmitter، ProximityPrompt
- معاينة ثلاثية الأبعاد للكائنات
- حفظ المثيلات والسكريبتات إلى ملف

### 📋 الخصائص (Properties)
- عرض وتعديل جميع خصائص الأجزاء في الوقت الفعلي
- إضافة وتعديل وحذف الخصائص المخصصة (Attributes)
- بحث متقدم في الخصائص
- إظهار الخصائص المخفية والمهملة
- دعم كامل لأنواع البيانات: Color3، NumberSequence، ColorSequence، إلخ.

### 💻 عارض السكريبتات (Script Viewer)
- عرض أكواد السكريبتات مع تنسيق (Syntax Highlighting)
- تفكيك (Decompile) السكريبتات يدعم جميع المنفذين تقريباً
- خيارات تفكيك احتياطية (Konstant API، Advanced Decompiler، Shiny)
- حفظ السكريبتات كملفات مصدر أو Bytecode
- نسخ إلى الحافظة وتنفيذ الكود

### ⌨️ الأوامر (Console)
- محطة أوامر تفاعلية
- سجل مخرجات مع تمرير تلقائي
- تحكم بحجم الخط
- مسح المخرجات

### 🖼️ معاينة ثلاثية الأبعاد (3D Viewer)
- عرض النماذج والقطع في نافذة معاينة تفاعلية
- تدوير تلقائي، تكبير/تصغير، قفل الكاميرا
- حفظ النماذج إلى ملف

### 💾 حفظ المثيلات (Save Instance)
- حفظ أي جزء في اللعبة إلى ملف
- خيارات متقدمة: تفكيك السكريبتات، حفظ المثيلات الفارغة، إزالة شخصيات اللاعبين
- دعم تعدد الخيوط (Multi-threading)

### ⚙️ الإعدادات (Settings)
- واجهة إعدادات شاملة
- تحكم بالمظهر والواجهة
- إعدادات المستكشف والخصائص
- إعدادات المفكك (Decompiler)
- حفظ واستعادة تلقائي للإعدادات

### 🎨 واجهة متطورة
- نافذة قابلة للسحب والتصغير والتكبير
- ألوان قابلة للتخصيص (Color Picker)
- منتقي الألوان المتقدم (RGB/HSV)
- محرر NumberSequence و ColorSequence
- قوائم منسدلة وأزرار ومربعات اختيار
- دعم الخطوط العربية

---

## 🛠️ التوافق

- يدعم جميع منفذي الأوامر الرئيسيين (Synapse، Script-Executor، إلخ)
- آلية احتياطية للدوال غير المدعومة (decompile, getscriptbytecode, saveinstance)
- يعمل في بيئات روبلوكس المختلفة
- دعم وضع عدم الاتصال (Offline Mode) عبر الوحدات المضمنة
